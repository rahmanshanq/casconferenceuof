# Central Asian Studies Conference 2026

**Voices through Art & Culture: Identity Formation in Central Asia, from Music to Architecture**

The official website for the inaugural Central Asian Studies Conference at the University of Chicago, April 17-18, 2026.

Organized by the Central Asian Studies Society.

## Tech Stack

- [SvelteKit 5](https://svelte.dev/) - framework
- [Tailwind CSS](https://tailwindcss.com/) - styling

## Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Deployment

Deployed automatically via Cloudflare Pages on push to `main`.
